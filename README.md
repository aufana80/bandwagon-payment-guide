## 搬瓦工支持哪些付款方式

目前搬瓦工一共有四种付款方式：

- **Alipay（支付宝）**：手机扫码付款，人民币结算，最推荐
- **Unionpay（银联）**：国内银行卡直接刷，或走网银
- **PayPal（贝宝）**：需要有 PayPal 账户
- **Credit Card（信用卡）**：Visa / Mastercard 双币卡

对国内用户来说，优先顺序大概是：支付宝 > 银联 > PayPal > 信用卡。支付宝扫码最快，银联刷卡也不麻烦，这两个选一个就行了。

## 搬瓦工支付宝付款怎么操作

流程很简单，说慢一点：

1. 选好套餐，加入购物车
2. 在结算页填写注册信息（邮箱必须真实，地址随便填城市就够了）
3. 付款方式选 **Alipay**
4. 勾选同意服务条款，点 **Complete Order**
5. 跳转到支付页后点 **Pay now**
6. 出现二维码，掏出手机扫一扫，完成

整个过程不用在网页上登录支付宝，也不用把支付宝账号告诉任何人——走的是扫码付款，跟你平时线下扫码买东西没太大区别。支付会通过 Payssion 的接口完成，这是专做海外收款的第三方支付平台，安全性没问题。

有一个细节要注意：**注册的时候不要开代理**。填写信息时挂着梯子，系统可能判定为欺诈然后自动退款。关掉代理再注册就好。

## 支付宝偶尔用不了怎么办

偶尔会遇到支付宝接口临时故障，跳出"系统有点儿忙"或者 SECONDARY_MERCHANT_STATUS_ERROR 这种错误。

这种时候不用着急，直接切换到**银联支付（Unionpay via Payssion）**。

银联支付的操作也很直接：
1. 在支付方式里选 Unionpay
2. 跳转到银联官网
3. 输入国内银行卡号，验证一下就完成了

如果习惯用网银，页面下方还有"网银支付"选项，可以直接跳到对应银行的页面完成付款，几乎所有主流国内银行都支持。很多银行 APP 还可以扫码，比在电脑上操作还快。

## 注册信息怎么填

这也是一个高频问题。全英文表格，有些人会纠结到底怎么填：

- **邮箱**：必须真实，这是你的登录账号，服务器信息也发到这里
- **姓名**：随便填，不做实名验证
- **国家**：选中国（China）
- **省/市**：随便填一个城市就行
- **手机号**：不用真实的
- **密码**：建议稍微设复杂一点，大小写加数字

填完之后选支付方式，走完支付流程，整个购买差不多 5 分钟以内。

## 常见问题 FAQ

**Q：搬瓦工用支付宝付款安全吗？**
A：安全。付款走扫码，不需要在网页登录支付宝账号，支付宝账号不会暴露给任何第三方。Payssion 是专做海外支付的正规接口，国内不少海外商家都在用。

**Q：支付宝付款时提示系统繁忙怎么办？**
A：改用银联支付（Unionpay）。国内所有银行的储蓄卡、信用卡都是银联卡，直接输卡号就行。也可以选网银支付，跳转到具体银行页面操作。订单通常保留 24 小时，不用重新下单。

**Q：搬瓦工可以月付吗？**
A：大部分套餐支持年付、半年付、季付。KVM 入门系列只有年付。CN2 GIA-E 支持季付，香港和日本高端套餐支持月付。具体要看结算页面的选项。

**Q：买了之后不满意可以退款吗？**
A：支持退款，但没有官方明确的"30天无理由退款"政策，退款需要联系客服处理，建议购买前先想清楚需求。

**Q：搬瓦工结算货币是美元还是人民币？**
A：搬瓦工所有套餐以美元标价，用支付宝付款时会按实时汇率换算成人民币，最终扣款金额会有小幅波动。
